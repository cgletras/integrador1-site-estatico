    Lógica de Programação

    O curso de Introdução a Lógica de Programação é fundamental para quem almeja atuar como programador de computadores. Os conhecimentos adquiridos no curso serão base para o aprendizado de outras linguagens de programação, tais como, Java, C, ou JavaScript. Um bom programador é aquele que, antes de conhecer uma linguagem específica, conhece a lógica por trás das operações.

    Em um primeiro momento este curso procura preparar o ambiente para a produção lógica. Esse preparo começa pelo ambiente do computador onde o desenvolvimento da logica tomará forma, chegando a mente do aluno de onde nascerá a logica. Esse preparo tende a abrir os caminhos cognitivos tão importantes no desenvolvimento de uma programador capaz de solucionar problemas e propor soluções dinamicas.
