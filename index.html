    Nossos Cursos

    Arq. e Organização de Computadores

    Entenda a arquitetura de um computador. O termo 'Arquitetura de um computador' refere-se aos atributos de um sistema que são visíveis para o programador. Já o termo 'Organização de um computador' refere-se as unidades operacionais e suas interconexões que implementam as especificações da sua arquitetura.

    Fundamentos de Web Design I

    Seja um desenvolvedor de websites, interfaces para sistemas web e páginas na internet de todo tipo, desde redes sociais e páginas de empresas até lojas virtuais. Com este curso, você aprenderá a utilizar as principais ferramentas para a criação e o desenvolvimento de websites. Você terá o conhecimento necessário para elaborar projetos e executar as atividades que a área necessita.

    Inglês Instrumental

    Inglês instrumental é uma metodologia de aprendizado de uma habilidade específica de forma isolada baseado no (E.S.P.) English for Specific Purpose, que em português quer dizer: Inglês com Objetivos Específicos. No Brasil a habilidade de leitura é a mais utilizada pelo Inglês Instrumental para tradução e interpretação de textos em inglês de forma rápida e efetiva.

    Lógica de Programação

    O curso de Introdução a Lógica de Programação é fundamental para quem almeja atuar como programador de computadores. Os conhecimentos adquiridos no curso serão base para o aprendizado de outras linguagens de programação, tais como, Java, C, ou JavaScript. Um bom programador é aquele que, antes de conhecer uma linguagem específica, conhece a lógica por trás das operações.

    Metodologia Científica

    Metodologia científica é o estudo dos métodos ou dos instrumentos necessários para a elaboração de um trabalho científico. É o conjunto de técnicas e processos empregados para a pesquisa e a formulação de uma produção científica.

    Sistemas Operacionais

    Os sistemas operacionais são os softwares básicos de um computador. Neste curso, será aprofundado aspectos que facilitarão a execução de tarefas que o auxiliarão a instalar, configurar e gerenciar dispositivos de hardware e software, bem como gerenciar arquivos e pastas
